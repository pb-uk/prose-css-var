Simpler prose styles for Tailwind CSS

The modern trend is towards soft resets like normalize-css and modern-normalize, however ‘hard’ resets like the original Meyerweb reset still have their uses, and in particular the surge in popularity of Tailwind CSS has increased their use. This page is concerned with Tailwind, if you are using another reset you should read this page instead.

Tailwind performs a ‘hard’ reset which removes things like margins from paragraphs, markers from lists, and even font sizes from headings, leaving your content almost impossible to read. prose-css-var fixes this when you wrap your text in a tag with the .prose css class. Also because prose-css-var uses css variables:

  • It does not work on IE11
  • It is easy to override colours for links, blockquote borders etc
  • You get dark mode and other theming abilities for free!
